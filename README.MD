# 🏥 Clínica San Francisco de Asís - Chachapoyas

Official repository for the web platform of **Clínica San Francisco de Asís**, located in Chachapoyas, Amazonas, Peru. This project is designed to provide a modern, fast, and accessible interface for our patients.

---

## 🚀 Project Overview
This website has been developed using a clean and responsive architecture to ensure patients can easily book appointments or consult medical services from any device through medinet as saas.

### **Tech Stack:**
* **Frontend:** HTML5, CSS3 (Sass), JavaScript.
* **Frameworks:** Bootstrap 5.3.
* **UI Libraries:** AOS (Animate on Scroll), GLightbox, Swiper.
* **Deployment:** GitHub Pages.

---

## 🛠️ Project Structure
* `/assets`: Contains images, styles (CSS/SCSS), and custom scripts.
* `/vendor`: External libraries (Bootstrap, FontAwesome, etc.).
* `/forms`: Contact and appointment form processing.
* `index.html`: Main landing page for the clinic.

## 👨‍💻 Development & Maintenance
This repository is managed by the clinic's administration and technology team.

**To update the site (Authorized Maintainers only):**
1. Sync the local repository with the `main` branch.
2. Implement and test changes locally.
3. Push to `main`; GitHub Actions will automatically trigger the live deployment.
